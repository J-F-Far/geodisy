Setting up GDAL

Install Proj 6.0.0
1) Download proj-6.0.0.tar.qz
2) Open Tar in /opt/source/
3) cd  proj-6.0.0
4) mkdir build
5) cd build
6) cmake ..
7) cmake --build .

Install Python3
Install Anaconda for Python3.x
Install Miniconda for Python3.x

Make a Symbol link to conda

conda install -c conda-forge gdal
